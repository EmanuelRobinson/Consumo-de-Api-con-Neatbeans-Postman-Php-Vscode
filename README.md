## Herramientas Necesarias
Para completar esta tarea, asegúrate de tener instaladas las siguientes herramientas:
* **XAMPP Control Panel** (Para el servidor local y base de datos).
* **Apache NetBeans** (Preferiblemente con **JDK 18** o superior).
* **Postman** (Para testear los endpoints de la API).
* **Visual Studio Code** (Para edición de código y frontend).
## Configuración del Entorno
Antes de empezar con el código, debemos preparar nuestra base de datos:
1.  Abre **XAMPP Control Panel**.
2.  Habilita (**Start**) los módulos de **Apache** y **MySQL**.
3.  En la fila de MySQL, haz clic en el botón **Admin**.
4.  Una vez en *phpMyAdmin*, crea una nueva base de datos con el nombre de tu preferencia.
¡Listo! Con esto tenemos los cimientos de nuestra aventura.
