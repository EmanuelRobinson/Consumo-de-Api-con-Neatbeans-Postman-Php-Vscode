<h1>Holaa</h1>
<hr>

<p>En este repositorio encontrarás un ejercicio práctico y corto para aprender a crear una API y realizar su consumo.</p>

<h2>Herramientas Necesarias</h2>
<p>Para completar esta tarea, asegúrate de tener instaladas las siguientes herramientas:</p>
<ul>
    <li><strong>XAMPP Control Panel</strong> (Para el servidor local y base de datos).</li>
    <li><strong>Apache NetBeans</strong> (Preferiblemente con <strong>JDK 18</strong> o superior).</li>
    <li><strong>Postman</strong> (Para testear los endpoints de la API).</li>
    <li><strong>Visual Studio Code</strong> (Para edición de código y frontend).</li>
</ul>

<h2>Configuración del Entorno</h2>
<p>Antes de empezar con el código, debemos preparar nuestra base de datos:</p>
<ol>
    <li>Abre <strong>XAMPP Control Panel</strong>.</li>
    <li>Habilita (<strong>Start</strong>) los módulos de <strong>Apache</strong> y <strong>MySQL</strong>.</li>
    <li>En la fila de MySQL, haz clic en el botón <strong>Admin</strong>.</li>
    <li>Una vez en <em>phpMyAdmin</em>, crea una nueva base de datos con el nombre de tu preferencia y con esto tenemos los cimientos de nuestra aventura.</li>
</ol>
